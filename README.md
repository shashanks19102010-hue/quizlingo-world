# 🌍 QuizLingo World 🎮📚

**Learn the world, one level at a time.**

QuizLingo World is a fun and interactive learning platform inspired by the game-like experience of Duolingo.
It is designed for students from **Class 1 to 12**, turning study sessions into exciting quiz adventures.

The main goal of this project is to make learning feel less like traditional homework and more like a game.
Students can choose their **language, class, and subject**, then progress through **50 engaging levels**, where each level includes **30 quiz challenges**.

With smooth animations, rewarding sounds, cute mascots, streaks, hearts, XP, and unlockable rewards, the platform keeps students motivated while helping them learn in a more enjoyable way.

This project is being built as a **global learning website**, so children from different countries can use it in their own language with a friendly and colorful experience.

---

## 🚀 Key Features

### 🌍 Global Learning Experience

* Supports multiple languages
* Language selection on first launch
* Flag-based language UI
* Friendly for kids worldwide

### 🎓 Smart Learning System

* 12 school classes
* Multiple subjects
* 50 levels for every subject
* 30 quizzes in each level
* Placeholder quiz system ready for future real data

### 🎮 Game-Style Motivation

* XP rewards
* Daily streaks
* Hearts/lives system
* Reward chests
* Boss levels
* Level unlock progression

### 🎨 Premium UI / UX

* Duolingo-inspired level path
* Mobile-first responsive design
* Smooth transitions and animations
* Cute animated mascots
* Fun sound effects and feedback

### 🔊 Interactive Sound System

* Click sound on every tap
* Correct answer sound
* Wrong answer feedback
* Level completion sound
* Background music loop
